# Homebrew Tap for Lua 5.1 (Legacy) - required for neovim compatability

Original formulae from [`homebrew-core`](https://github.com/Homebrew/homebrew-core/tree/master), based off [`commit`](https://github.com/Homebrew/homebrew-core/commit/3c112b2e221ad518c1468e7be7f13adf85dd1530)

### Adding Tap

```
brew tap a-t-eight/homebrew-lua-5.1
```

### Installing Lua 5.1

```
brew install a-t-eight/lua-5.1/lua
```
